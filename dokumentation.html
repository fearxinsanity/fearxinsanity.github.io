Projektdokumentation

Das Projekt „Das unberechenbare Volk“ ist ein interaktiver Wahl-Simulator zur Visualisierung von Wahldynamiken. Die Anwendung wird als Windows-Anwendung mit einer grafischen Benutzeroberfläche (GUI) umgesetzt und soll die Unvorhersehbarkeit politischer Prozesse durch Zufallsereignisse modellieren.

Aufbau der Simulation

Die Systemarchitektur folgt dem Model-View-Controller (MVC)-Design-Pattern. Dies trennt die Simulationslogik (`Model`) von der Benutzeroberfläche (`View`) und dem Vermittler (`Controller`). Der Datenfluss ist unidirektional: Nutzereingaben in der `View` werden vom `Controller` verarbeitet und an das `Model` weitergeleitet. Das `Model` führt die Simulation aus und der `Controller` aktualisiert die `View` mit den neuen Daten. Die Simulation ist zeitabhängig und läuft in diskreten Zeitschritten ab.

Genutzte Technologien

  • Programmiersprache: Java SE
  • Frameworks/Bibliotheken: JavaFX für das UI und Maven zur Abhängigkeitsverwaltung.
  • Java Development Kit (JDK): JDK 21.
